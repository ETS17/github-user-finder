Как запустить?

Установить Python
Открыть терминал и ввести:
запросы на установку pip

Запустить файл:
python github_user_finder.py

Как пользоваться?

Введите имя пользователя в поле поиска
Нажмите «Найти»
Выберите пользователя из списка
Нажмите «В избранное» — он сохранится
Нажмите «Показать избранное» — чтобы увидеть всех сохраненных пользователей
Какой API используется

Адрес GitHub REST API: https://api.github.com/search/users

Пример работы

Поиск: Ввод: octocat → Находит пользователя

Ошибки:

Пустое поле → «Введите имя пользователя»
Нет интернета → «Ошибка подключения»
